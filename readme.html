Controls

HOLD acts as the right mouse button; DRAG acts as the joystick, controlling character movement and menu navigation;

Cancel acts as the back button to bring up the menu. In some games, you may need to click this after loading a save to regain control;

↵ is the Enter key, used to enter the game and confirm selections;

⚙️ opens other functions, such as toggling fullscreen (hiding buttons), toggling mouse operations, and using F8 to change display info.

Note: downloading the Save folder, loading to the Save folder, and downloading the license are not supported in a webview environment. These functions are only available when accessed via a browser with an active server port.
